# AI Fitness App

This is a simple AI-powered fitness app built with Streamlit. It helps users plan their diet and exercise based on their weight goals.

## Features

- Input daily food intake
- Input current weight, height, age, gender
- Specify weight loss or gain goal
- Get personalized diet recommendations
- Get weekly exercise schedule
- Estimate time to achieve weight goal

## Installation

1. Install Python (if not already installed).
2. Clone or download this repository.
3. Install dependencies: `pip install -r requirements.txt`

## Usage

Run the app with: `streamlit run app.py`

Open the provided URL in your browser and fill in the inputs.

## Troubleshooting

- Ensure all inputs are filled before clicking "Get Recommendations".
- For accurate results, provide realistic values.
- This is a basic app; consult a professional for personalized advice.
